# Dlavie Engine WA

Dlavie Engine WA adalah fondasi bot WhatsApp multi-device berbasis Baileys dengan pairing code, command router modular, security layer dasar, cooldown, dan branding Dlavie di setiap bagian utama engine.

## Fitur Dlavie

- Dlavie multi-device WhatsApp connection
- Dlavie pairing code login
- Dlavie modular command loader
- Dlavie owner guard
- Dlavie self/public mode
- Dlavie cooldown untuk mengurangi spam
- Dlavie session folder

## Instalasi Dlavie

```bash
npm install
cp .env.dlavie.example .env.dlavie
```

Jalankan Dlavie Engine:

```bash
npm run dlavie:start
```

Masukkan pairing code dari terminal ke WhatsApp melalui Linked Devices.

## Command Dlavie Awal

- `.menu`
- `.ping`
- `.runtime`
- `.owner`
- `.echo teks`
